# Android-NewsApp
  
Implmented a news aggregator for news articles from Guardian news source. The app provides multiple functionalities like search functionality to search for relevant articles, bookmark favorite articles, redirect articles to parent website, search for google trends for different words across the internet and share articles on social media platforms.

Server-Side:  
JavaScript  
NodeJS  
Fetch API  
CORS Module  
  
Android-App:  
Android Studio  
Java  
Volley  
Picasso  
Bing Autosuggest API  
Google Trends API
